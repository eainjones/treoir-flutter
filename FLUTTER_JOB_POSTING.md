# Flutter iOS Developer - Workout Logging App

> Contract role | 6-8 weeks | Remote

---

## The Project

Build a native iOS workout logging app using Flutter. The backend REST API and web app already exist - you're building the mobile client only.

**App Purpose:** Log gym workouts (sets, reps, weight) with a clean, fast interface optimized for use between sets.

---

## Tech Stack

- **Flutter 3.x** (iOS target, Android later)
- **Supabase** for auth and realtime
- **State management:** Riverpod or Bloc (your preference)
- REST API consumption (spec provided)

---

## Scope

### In Scope
- Auth (email, Apple Sign-In)
- Start/complete workouts
- Add exercises, log sets
- Rest timer
- Workout history
- Routine templates
- Settings (kg/lbs units)

### Out of Scope
- Backend development
- Web app
- Android (for now)
- Offline-first sync
